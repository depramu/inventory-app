# Inventory App Frontend

This project is a **simple SaaS** (Software as a Service) frontend for inventory and transaction management.

## Features

- Authentication: login and register
- Dashboard with KPI cards and 7-day chart
- Product management: add, update, delete
- Product tools: search, sorting, low stock filter
- Transaction creation and transaction history
- SaaS-style UI with fixed sidebar navigation

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript
- REST API integration

## Notes

- This app is designed as a lightweight SaaS-style interface for learning and small internal use.
- API base URL is configured in `script.js`.
